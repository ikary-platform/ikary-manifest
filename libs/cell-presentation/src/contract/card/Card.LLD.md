# Card LLD

Renders `<div className="rounded-lg border bg-card text-card-foreground shadow-sm">`.
Header (shown when title or description present): `<div className="flex flex-col space-y-1.5 p-6">`.
Content (shown when content present): `<div className="p-6 pt-0">`.
Footer (shown when footer present): `<div className="flex items-center p-6 pt-0">`.
No Radix dependency — pure HTML/Tailwind.
