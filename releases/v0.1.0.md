# v0.1.0

First public release of `@ikary/cli` on npm.

## Added

- Published CLI as `@ikary/cli` — install globally with `npm install -g @ikary/cli` or run with `npx @ikary/cli`
- `ikary init` — scaffold a new Cell manifest in the current directory
- `ikary validate` — validate a manifest against the IKARY schema
- `ikary compile` — compile a manifest to the runtime contract format
- `ikary preview` — open a self-contained HTML preview of a compiled manifest
- `ikary local start/stop/status/logs/reset-data/db` — manage the full local development stack
- `ikary primitive add/validate/list/studio` — scaffold and preview custom UI primitives
- `ikary setup ai` — configure Claude Code integration
