# v0.1.0

First public release of the ikary CLI. Install globally or run with npx to scaffold, validate, compile, and preview Cell manifests, and to manage the local development stack.

## CLI

- Published `@ikary/cli` to npm — available as `npm install -g @ikary/cli` or `npx @ikary/cli`
- Added `ikary init` — scaffold a new Cell manifest in the current directory
- Added `ikary validate` — validate a manifest against the IKARY schema
- Added `ikary compile` — compile a manifest to the runtime contract format
- Added `ikary preview` — open a self-contained HTML preview of a compiled manifest
- Added `ikary local start/stop/status/logs/reset-data/db` — manage the full local development stack
- Added `ikary primitive add/validate/list/studio` — scaffold and preview custom UI primitives
- Added `ikary setup ai` — configure Claude Code integration
