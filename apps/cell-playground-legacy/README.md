# @ikary/cell-playground-legacy

**Deprecated.** This directory is the former `apps/cell-playground` internal
dev tool. It is kept temporarily because `apps/cell-playground` (the public
showcase, formerly `apps/playground-app`) has not yet reached feature parity.

Once the public `cell-playground` covers every feature that lived here,
this directory will be deleted in a follow-up PR. Do not add new features
here; add them to `apps/cell-playground/` instead.

## Running

```bash
pnpm --filter @ikary/cell-playground-legacy dev
```

Runs on port 4504.
