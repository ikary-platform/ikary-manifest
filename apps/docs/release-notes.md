# Release Notes

IKARY publishes release notes for the CLI and user-facing package changes.

## Primary sources

- GitHub Releases: [ikary-platform/ikary-manifest releases](https://github.com/ikary-platform/ikary-manifest/releases)
- Repository release files: [releases/ directory](https://github.com/ikary-platform/ikary-manifest/tree/main/releases)

## How to read release notes

1. Start with the latest CLI release section.
2. Check package sections that affect your runtime path.
3. Apply upgrade changes in a feature branch and run `ikary validate` before deployment.
